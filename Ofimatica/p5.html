¿ Que es la maquetación ?


La maquetación es la forma de poner los textos, imagenes y espacios de forma estandar para generar un espacio especifico. Por ejemplo la maquetación WEB hace referencia a como se disponen los elementos de la página según el tipo y la necesidad. A continuación se presentan dos software de maquetación que se pueden aplicar a objetos reales.


Scribus y procesadores de texto con plantillas

El software scribus permite hacer la maquetación de elementos como libros, en elementos como sus margenes, separación entre las lineas de los parrafos entre otros.



En ocasiones para los software procesadores de textos se fabrican plantillas con tamaños determinados que permiten generar los textos así que la imagen anterior se pueden desarrollar tanto con scribus como con las plantillas.
