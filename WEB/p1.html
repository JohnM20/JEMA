Desarrollo WEB y HTML


La tecnología WEB es una rama de la tecnología que funciona en redes donde diversos dispositivos que funcionan con información digital general la comparten. La red de conexión de información más grande del mundo denominada internet, y las redes más pequeñas que no se conectan con las demas pero si para un uso privado se llaman intranet .



La imagen anterior es una página WEB que muestra información tanto de texto como imagenes, y es el desarrollo del curso. En este modulo se trabajara el concepto de la clasificación de las redes, así como la diferencia entre dos conceptos de diseño WEB denominados backend y frontend así como las principales tecnologías basadas en cada concepto. Para finalizar se realizara un acercamiento de la tencología HTML y el Arból DOM como elemento base de la tecnología WEB.


Árbol DOM



El árbol DOM es la estructura jerarquica de una página WEB, la mayoria de elementos WEB se encuentran dentro de etiquetas como se muestra en la imagen anterior, y según este conformado un elemento dentro de otro elemento se genera lo que visualmente denominamos página WEB, así también modificamos su contenido.
