Desarrollo WEB y HTML


La tecnología WEB es una rama de la tecnología que se basa en la red de conexión de información más grande del mundo denominada internet, y aunque puede funcionar en redes más pequeñas esta tecnología es primordial para redactar y leer masivamente información.



La imagen anterior es una página WEB que muestra información tanto de texto como imagenes se denomina una página WEB, y es el desarrollo de la misma el foco del curso. Si bien existen páginas que permiten el envío de información y mostrar información de forma dinamica. Este curso solo trabajara la parte estetica denominada frontend y algunos indicios acerca de la conexión de los datos adquiridos.


Árbol DOM



El árbol DOM es la estructura jerarquica de una página WEB, la mayoria de elementos WEB se encuentran dentro de etiquetas como se muestra en la imagen anterior, y según este conformado un elemento dentro de otro elemento se genera lo que visualmente denominamos página WEB, así también modificamos su contenido.
