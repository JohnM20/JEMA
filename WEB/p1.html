Estructura WEB


Si nos movemos por la ciudad de un lugar a otro es habitual tener un dirección que identifica donde estamos y hacia donde queremos llegar. El funcionamiento de la estructura WEB es muy similar, al momento que necesitamos acceder a algún tipo de información que no tenemos en nuestros equipos digitales como el celular, computador, etc. Para encontrar esta información accedemos desde una aplicación o un navegador WEB (dirección donde estamos) a la red donde la conseguimos (dirección donde esta la información).



Al usar una aplicación como una red social de forma interna se conecta a una dirección que en este caso es virtual y es invisible para nosotros, cuando usamos un navegador WEB como Internet Explorer, Google Chrome o Firefox accedemos desde la URL como la que se encuentra en la imagen anterior a la dirección electrónica de la página que tiene el servidor.


Si queremos ingresar a una página como www.hazlotec.com tambien podriamos hacerlo con una serie de números como 18.65.229.18 sin embargo seria mucho más dificil de recordar, así toda página tiene un dominio WEB que significa una serie de números y letras denominadas dirección IP que identifican para una red nuestro equipo y de a donde sacamos la información quedando siempre registrado por nuestro proveedor de internet.


Para una red de internet siempre vamos a tener una IP asignada por nuestro proovedor de datos moviles o de internet de nuestra casa y va a cambiar cada vez que nos conectemos por la dinamica propia de la red. Pero para una conexión más pequeña y más privada que incluso no maneje internet es posible tener una dirección IP fija todo el tiempo, así las redes se clasifican de la siguiente manera y en todas es posible trabajar con tecnología WEB:

Donde las redes LAN se denominan locales y las redes WAN se denominan extensas, y en este entorno funciona todo lo que conocemos para compartir la información reconociendo que existen diferentes aplicaciones WEB que no se basan necesariamente en el uso del internet. Para que exista una red existen dos elementos básicos, un servidor que es el encargado de albergar la red y un cliente que es el encargado de solicitar la información.
