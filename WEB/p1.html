Desarrollo WEB y HTML


El desarrollo WEB es una rama de la tecnología que funciona con dispositivos que comparten información digital. La forma como se organizan los diferentes equipos para compartir la información se denominan redes, entre las que se encuentra la red más grande conocida denominada internet. Como se muestra la información y se vuelve interactiva se denomina desarrollo WEB, un ejemplo es la siguiente imagen donde podemos visualizar una estructura de página que muestra información tanto de texto como imagenes, sin embargo el desarrollo WEB permite involucrar una infinidad de elementos multimedia.



En este modulo se abordara el funcionamiento de las redes que albergan la tecnología WEB, así como la diferencia entre dos conceptos de diseño denominados backend y frontend, con algunas de sus principales tecnologías. Para finalizar se realizara un acercamiento de la tecnología HTML y el Arból DOM como elemento base de la tecnología WEB.


Estructura WEB

+

Si nos movemos por la ciudad de un lugar a otro es habitual tener un dirección que identifica donde estamos y hacia donde queremos llegar. El funcionamiento de la estructura WEB es muy similar, al momento que necesitamos acceder a algún tipo de información que no tenemos en nuestros equipos digitales como el celular, computador, etc. Para encontrar esta información accedemos desde una aplicación o un navegador WEB (dirección donde estamos) a la red donde las conseguimos (dirección donde esta la información).



Al usar una aplicación como una red social de forma interna se conecta a una dirección que en este caso es virtual y es invisible para nosotros, cuando usamos un navegador WEB como Internet Explorer, Google Chrome o Firefox accedemos desde la URL como la que se encuentra en la imagen anterior a la dirección electrónica de la página que tiene el servidor. Si quieremos ingresar a una página como www.hazlotec.com tambien podriamos hacerlo con una serie de números como 18.65.229.18 sin embargo seria mucho más dificil de recordar, así toda página tiene un dominio WEB que significa una serie de números y letras denominadas dirección IP que identifican para una red nuestro equipo y de a donde sacamos la información quedando siempre registrado por nuestro proveedor de internet.


Para una red de internet siempre vamos a tener una IP asignada por nuestro proovedor de datos moviles o de internet de nuestra casa y va a cambiar cada vez que nos conectemos por la dinamica propia de la red. Pero para una conexión más pequeña y más privada que incluso no maneje internet es posible tener una dirección IP fija todo el tiempo, así las redes se clasifican de la siguiente manera y en todas es posible trabajar con tecnología WEB:

Donde las redes LAN se denominan locales y las redes WAN se denominan extensas, y en este entorno funciona todo lo que conocemos para compartir la información reconociendo que existen diferentes aplicaciones WEB que no se basan necesariamente en el uso del internet. Para que exista una red existen dos elementos básicos, un

servidor

que es el encargado de albergar la red y un cliente que es el encargado de solicitar la información.

Backend y Frontend

+

Consiste en una secuencia de pasos ordenados, por ejemplo, cuando vamos a usar el televisor podemos seguir un algoritmo como el siguiente:


1. Buscar el control remoto
2. Prender el televisor
3. Encontrar el canal preferido
4. Apagar el televisor
5. Guardar el control remoto

Así los pasos pueden variar según se desconecte el TV o usemos un servicio de streaming, siendo así la esencia de los algoritmos donde un solo problema puede tener múltiples soluciones.



Árbol DOM



El árbol DOM es la estructura jerarquica de una página WEB, la mayoria de elementos WEB se encuentran dentro de etiquetas como se muestra en la imagen anterior, y según este conformado un elemento dentro de otro elemento se genera lo que visualmente denominamos página WEB, así también modificamos su contenido.
