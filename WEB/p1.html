Desarrollo WEB y HTML


El desarrollo WEB es una rama de la tecnología que funciona con dispositivos que comparten información digital. La forma como se organizan los diferentes equipos se denominan

redes

entre las que se encuentra la red más grande conocida denominada internet. Como se muestra la información se denomina tecnología WEB, un ejemplo es la siguiente imagen donde podemos visualizar una estructura de página que muestra información tanto de texto como imagenes, sin embargo el desarrollo WEB permite involucrar una infinidad de elementos multimedia.



En este modulo se abordara la clasificación de las redes que albergan la tecnología WEB, así como la diferencia entre dos conceptos de diseño denominados backend y frontend, así como las principales tecnologías basadas en cada concepto. Para finalizar se realizara un acercamiento de la tecnología HTML y el Arból DOM como elemento base de la tecnología WEB.

Consiste en una secuencia de pasos ordenados, por ejemplo, cuando vamos a usar el televisor podemos seguir un algoritmo como el siguiente:


1. Buscar el control remoto
2. Prender el televisor
3. Encontrar el canal preferido
4. Apagar el televisor
5. Guardar el control remoto

Así los pasos pueden variar según se desconecte el TV o usemos un servicio de streaming, siendo así la esencia de los algoritmos donde un solo problema puede tener múltiples soluciones.



Árbol DOM



El árbol DOM es la estructura jerarquica de una página WEB, la mayoria de elementos WEB se encuentran dentro de etiquetas como se muestra en la imagen anterior, y según este conformado un elemento dentro de otro elemento se genera lo que visualmente denominamos página WEB, así también modificamos su contenido.
