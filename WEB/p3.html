Bootstrap


Es una libreria que añade estilos a las etiquetas usando el lenguaje CSS, y entre uno de sus mayores aportes es generar páginas adaptables a todas las pantallas, para que funcionen es necesario agregar al documento HTML algunas lineas de codigo, a continuación se presentan las lineas junto a algunas etiquetas fundamentales de HTML, para facilitar la construcción de codigo: 


			
			<!DOCTYPE html>
			<html lang="es">
			<head>
			<meta charset="UTF-8">
			<meta name="viewport" content="width=device-width, initial-scale=1.0">
			<title>Documento con Bootstrap</title>
			
			<!-- Agregar el enlace al archivo CSS de Bootstrap -->
			<link href="https://stackpath.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css" rel="stylesheet">
			
			</head>
			<body>
			
			<!-- Tu contenido HTML aquí -->
			
			<!-- Agregar los scripts de Bootstrap (jQuery y Popper.js son requeridos) -->
			<script src="https://code.jquery.com/jquery-3.5.1.slim.min.js"></script>
			<script src="https://cdn.jsdelivr.net/npm/@popperjs/core@2.9.3/dist/umd/popper.min.js"></script>
			<script src="https://stackpath.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"></script>
			
			</body>
			</html>

Contenedores y columnas


Para que funcionen las columnas se debe poner el atributo class="container"a una división y dentro de ellas otra división con el atributo class="row", a continuación se presenta como se construyen dos columnas:


Código Resultado 
		
			<div class="container">
			  <div class="row">
			    <div class="col-6">
			      Primer columna
			    </div>
			    <div class="col-6">
			      Segunda columna
			    </div>
			  </div>
Primer columna
Segunda columna

Así dependiendo el número de columnas que se quieran se debe indicar en el número despues de col- la cantidad de espacio ocupado sea el total 12. Así en el ejemplo anterior dos columnas cada una equivale 6, si se quisieran 3 columnas el numero a escribir seria col-4 como podemos ver en el siguiente ejemplo:

Código Resultado 
		
			<div class="container">
			  <div class="row">
			    <div class="col-4">
			      Primer columna
			    </div>
			    <div class="col-4">
			      Segunda columna
			    </div>
			    <div class="col-4">
			      Tercer columna
			    </div>
			  </div>

Navbar


Hace referencia a las tablas de navegación de una página WEB como lo son los menus de las páginas WEB, y son por lo general los que determinan la navegación de los sitios WEB


Código Resultado 
		
<nav class="navbar ">
    <ul class="nav navbar-nav">
    <li class="nav-item">
        <a class="nav-link" href="#">Logo</a>
    </li>
    <li class="nav-item">
        <a class="nav-link" href="#">Acerca</a>
    </li>
    <li class="nav-item">
        <a class="nav-link" href="#">Contacto</a>
    </li>
   
</ul>
</nav>

Formularios


Los formularios para tomar datos de la página WEB, tambien se pueden hacer con bootstrap, a continuación se presenta un ejemplo con diversos ejemplos:


Código Resultado 
		
<form>
      <!-- Campo de entrada -->
      <div class="mb-3">
        <label for="nombre" class="form-label">Nombre</label>
        <input type="text" class="form-control" id="nombre" placeholder="Ingrese su nombre">
      </div>
      <!-- Casilla de verificación -->
      <div class="mb-3 form-check">
        <input type="checkbox" class="form-check-input" id="acepto">
        <label class="form-check-label" for="acepto">Acepto los términos y condiciones</label>
      </div>
      <!-- Botones de radio -->
      <div class="mb-3">
        <div class="form-check">
          <input class="form-check-input" type="radio" name="opciones" id="opcion1" value="opcion1" checked>
          <label class="form-check-label" for="opcion1">
            Opción 1
          </label>
        </div>
        <div class="form-check">
          <input class="form-check-input" type="radio" name="opciones" id="opcion2" value="opcion2">
          <label class="form-check-label" for="opcion2">
            Opción 2
          </label>
        </div>
      </div>
      <!-- Selector -->
      <div class="mb-3">
        <label for="select" class="form-label">Seleccione una opción</label>
        <select class="form-select" id="select">
          <option selected>Seleccionar...</option>
          <option value="opcion1">Opción 1</option>
          <option value="opcion2">Opción 2</option>
          <option value="opcion3">Opción 3</option>
        </select>
      </div>
      <!-- Botón de envío -->
      <button type="submit" class="btn btn-primary">Enviar</button>
    </form>


Medidas absolutas


En el desarrollo WEB es indispensable el manerjo de medidas tanto absolutas como relativas, así se pueden hacer diseño respecto a unos centimetros en especifico o algunos pixeles en particular así también existen las pulgadas, milimetros y demas medidas asociadas que es indispensable manejar y convertir entre ellas.



Medidas relativas

En ocasiones es necesario fijar las medidas, según cambia el tamaño de la pantalla o el tipo de dispositivos para este tipo de medidas se usan las tipo em, rem así como los porcentajes.
