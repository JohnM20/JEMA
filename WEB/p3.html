Boostrap


Es una libreria que añade estilos a las etiquetas usando el lenguaje CSS, y entre uno de sus mayores aportes es generar páginas adaptables a todas las pantallas, para que funcionen en la parte de Head se declaran 5 lineas de codigo con el link de Bootstrap, una libreria de Jquery que revisaremos más adelante, una libreria llamada pooper.js


Contenedores y columnas


Para que funcionen las columnas se debe poner el atributo class="container"a una división y dentro de ellas otra división con el atributo class="row", en donde se pueden hacer columnas que sean multiplos de doce (2 de 6 , 3 de 4 o 4 de 3)

Medidas absolutas


En el desarrollo WEB es indispensable el manerjo de medidas tanto absolutas como relativas, así se pueden hacer diseño respecto a unos centimetros en especifico o algunos pixeles en particular así también existen las pulgadas, milimetros y demas medidas asociadas que es indispensable manejar y convertir entre ellas.



Medidas relativas

En ocasiones es necesario fijar las medidas, según cambia el tamaño de la pantalla o el tipo de dispositivos para este tipo de medidas se usan las tipo em, rem así como los porcentajes.
