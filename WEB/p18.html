En la siguiente imagen podemos ver en la mitad un servidor el cual alberga unas tecnologías propias para su programación como lo es PHP,NodeJs,Ruby y se encarga de controlar los datos que se envian al servidor. Los computadores son los clientes los cuales visualizan la información y manejan tecnologías como HTML, Javascript, CSS.

Es importante entender que toda la programación que realicemos con la tecnología frontend se facilmente portable porque se ejecutan desde cualquier computador, sin embargo las tecnologías backend unicamente se programan en servidores así que es necesario volver un equipo servidor para poder programar unicamente en el o contratar un servidor para poder programar la gestión y recepción de datos. Actualmente desde las tecnologías frontend se realizan multiples procesos para no cargar los servidores gracias a las altas prestaciones computacionales de los equipos empleando tecnologías como node.



Arreglos


Los arreglos nos permiten guardar diferentes valores en una sola variable, las cuales se pueden agregar, quitar y recorrer entre muchas otras opciones que a continuación se presentan:

    

// Creación de un array vacío
let frutas = [];

// Añadir elementos al array usando push()
frutas.push("Manzana");
frutas.push("Banana");
frutas.push("Naranja");

// Acceder a elementos del array por índice
console.log("Primera fruta:", frutas[0]); // Salida: Manzana
console.log("Segunda fruta:", frutas[1]); // Salida: Banana

// Modificar un elemento del array
frutas[0] = "Pera";
console.log("Primera fruta modificada:", frutas[0]); // Salida: Pera

// Longitud del array
console.log("Cantidad de frutas:", frutas.length); // Salida: 3

// Recorrer el array usando un bucle for
console.log("Frutas:");
for (let i = 0; i < frutas.length; i++) {
    console.log("- " + frutas[i]);
}

// Eliminar el último elemento del array usando pop()
let ultimaFruta = frutas.pop();
console.log("Última fruta eliminada:", ultimaFruta); // Salida: Naranja
console.log("Frutas después de eliminar la última:", frutas); // Salida: ["Pera", "Banana"]
