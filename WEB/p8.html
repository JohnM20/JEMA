Cards

Las tarjetas o cards, permiten mostrar información de diferentes caracteristicas de las páginas WEB, a continuación se muestra el ejemplo de una:

Código Resultado 
		
	 <div class="container">
  	 <div class="card" style="width: 18rem;">
   	  <img src="https://via.placeholder.com/150" class="card-img-top" alt="Placeholder Image">
  	   <div class="card-body">
    	   <h5 class="card-title">Ejemplo tarjeta </h5>
     	  <p class="card-text">Aqui encontramos un ejemplo de una tarjeta </p>
     	  <a href="#" class="btn btn-primary">Go somewhere </a>
   	  </div>
 	  </div>
	 </div>

Jumbotron



El Jumbotron permite generar un contenido gráfico para generar interes en una página WEB como anuncio o una presentación del sitio, a continuación se presenta el respectivo codigo:


Código Resultado 
					
<div class="jumbotron">
<h1 class="display-4">¡Bienvenido!</h1>
<p class="lead">Este es un ejemplo .</p>
<hr class="my-4">
<p>Puedes utilizarlo para destacar contenido .</p>
 <a class="btn btn-primary btn-lg" href="#" role="button">Leer más</a>
</div>

¡Bienvenido!

Este es un ejemplo de Jumbotron de Bootstrap. Un Jumbotron es una gran caja de contenido para resaltar información clave.

Puedes utilizarlo para destacar contenido importante en tu página web.
