Asincronismo


En ocasiones es necesario realizar diversos codigos al mismo tiempo, para esta finalidad se emplea el concepto de asincronismo. Javascript genera las intrucciones y funciones según el programa se ejecute de orden secuencial, lo que quiere decir una tras otra cen un lugar llamado call stack, sin embargo en ocasiones una función puede demorar mucho esperando el servidor, o un evento del usuario. Así inicia el concepto de paralelismo donde si bien javascript no tiene una estructura para soportar multiples programas, tiene un event loop el cual es un inspector quien esta vigilando constantemente las funciones y tareas, en caso de que una se demore mucho le da paso a otras mientras termina, generando así una estructura de realizar estructuras paralelas. Así conociendo el funcionamiento del event loop es posible generar funciones con tiempo que se ejecutan de forma controlada.


Set time out y Set interval


Son funciones que permiten controlar el tiempo, set time out funciona programando el tiempo que se requiere que se ejecute la función a continuación se presenta un ejemplo: 


    

console.log('Inicio del programa');

// Función que simula una tarea asíncrona que tarda un segundo en completarse
function tareaAsincrona() {
  console.log('Inicio de la tarea asíncrona');
  setTimeout(function() {
    console.log('Fin de la tarea asíncrona');
  }, 1000); // Espera 1 segundo antes de ejecutar la función
}

// Llamamos a la función tareaAsincrona
tareaAsincrona();

console.log('Fin del programa');

Set interval repite una función las veces que se requiera en el codigo

    

// Definimos una función que se ejecutará cada segundo
function imprimirMensaje() {
  console.log('¡Hola mundo!');
}

// Llamamos a setInterval para ejecutar la función imprimirMensaje cada segundo (1000 milisegundos)
var intervalo = setInterval(imprimirMensaje, 1000);

// Después de 5 segundos, detenemos el intervalo
setTimeout(function() {
  clearInterval(intervalo); // Detenemos el intervalo
  console.log('Intervalo detenido después de 5 segundos.');
}, 5000);

Async - Await


Junto a la libreria Fecth async con away permiten hacer el proceso con peticiones de elementos a un servidor a continuación se presenta un ejemplo:


    

// Función que devuelve una promesa que se resuelve después de un cierto tiempo
function esperarUnSegundo() {
  return new Promise(resolve => {
    setTimeout(() => {
      resolve('Resuelto');
    }, 1000);
  });
}

// Función asincrónica que espera un segundo antes de imprimir el mensaje
async function imprimirMensaje() {
  const resultado = await esperarUnSegundo();
  console.log(resultado);
}

// Llamada a la función asincrónica
imprimirMensaje();
