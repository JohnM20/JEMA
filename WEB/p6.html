Funciones en Javascript


Las funciones en javascript permiten generar modulos de codigo que se pueden reutilizar en diferentes ocasiones, a continuación se presentan un ejemplo:


    
let nombre="John";
// Definición de una función
function saludar(nombre) {
    alert("¡Hola, " + nombre + "! Bienvenido.");
}

// Llamada a la función
saludar("Juan");

Asi en el ejemplo anterior solo es necesario cambiar el nombre en la función para generar diferentes saludos, la variable nombre, se llama parametro y es una variable interna del uso de las funciones.



Arreglos


Los arreglos nos permiten guardar diferentes valores en una sola variable

    

estudiantes=["Juan","Luis","Fabian","Elkin"]
for x in estudiantes:
  print(x)

Una utilidad de recorrer una lista es encontrar la posición de un elemento usando el comando index, a continuación se presenta el ejemplo empleando para encontrar la posición del estudiante fabian

    

estudiantes=["Juan","Luis","Fabian","Elkin"]
for x in estudiantes:
    posicion = estudiantes.index("Fabian");
print(posicion);
