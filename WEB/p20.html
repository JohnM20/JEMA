Evento Onclick y Modificar elementos por ID


Los eventos son las posibles iteracciones que tiene un usuario con una página WEB como hacer click en un elemento, mover las teclas del teclado, enviar un formulario o se cambia el tamaño de la pantalla entre otros. Javascript permite cambiar elementos del árbol DOM, como quitarlos, ponerlos o modificar su contenido. En el codigo siguiente podemos ver un uso clasico y practico de las dos caracteristicas anteriormente expuestas: 

    

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Modificar Texto con JavaScript</title>
</head>
<body>
  <h1 id="titulo">Texto Original</h1>
  <button onclick="cambiarTexto()">Cambiar Texto</button>

  <script>
    function cambiarTexto() {
      // Seleccionamos el elemento h1 por su id
      const tituloElemento = document.getElementById('titulo');

      // Modificamos el texto del elemento h1
      tituloElemento.textContent = '¡Nuevo Texto Modificado!';
    }
  </script>
</body>
</html>
