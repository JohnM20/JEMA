Evento Onclick y Modificar elementos por ID


Los eventos son las posibles iteracciones que tiene un usuario con una página WEB como hacer click en un elemento, mover las teclas del teclado, enviar un formulario o se cambia el tamaño de la pantalla entre otros. Javascript permite cambiar elementos del árbol DOM, como quitarlos, ponerlos o modificar su contenido. En el codigo siguiente podemos ver un uso clasico y practico de las dos caracteristicas anteriormente expuestas: 

    

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Modificar Texto con JavaScript</title>
</head>
<body>
  <h1 id="titulo">Texto Original</h1>
  <button onclick="cambiarTexto()">Cambiar Texto</button>

  <script>
    function cambiarTexto() {
      // Seleccionamos el elemento h1 por su id
      const tituloElemento = document.getElementById('titulo');

      // Modificamos el texto del elemento h1
      tituloElemento.textContent = '¡Nuevo Texto Modificado!';
    }
  </script>
</body>
</html>

Storage


Una forma de almacernar variables por mucho tiempo en una página WEB es usando Storage, a continuación se presenta un ejemplo; 

    

// Almacenar datos en localStorage
localStorage.setItem('nombre', 'Juan');
localStorage.setItem('edad', '30');

// Obtener datos de localStorage
let nombre = localStorage.getItem('nombre');
let edad = localStorage.getItem('edad');

// Mostrar los datos obtenidos
console.log("Nombre:", nombre); // Salida: Nombre: Juan
console.log("Edad:", edad); // Salida: Edad: 30

// Eliminar un dato de localStorage
localStorage.removeItem('edad');

// Mostrar los datos restantes
console.log("Datos restantes:", localStorage.getItem('nombre')); // Salida: Datos restantes: Juan
console.log("Edad después de eliminar:", localStorage.getItem('edad')); // Salida: Edad después de eliminar: null

JSON


Una forma de guardar datos para despues almacenar es el protocolo JSON que permite ajustar un protocolo de enviar los datos que se conecta a multiples tenoclogías a continuación se presenta un ejemplo:

Un contador permite llevar un control de cuantas veces se repite un ciclo y es util para codigos donde se debe llevar un conteo, por ejemplo si en este caso se quiere llevar un conteo de cuantas veces fallo al momento de salir del parqueadero


    
// Definición de un objeto JSON
let persona = {
    "nombre": "Juan",
    "edad": 30,
    "ciudad": "Madrid"
};

// Convertir el objeto JSON a una cadena JSON
let jsonStr = JSON.stringify(persona);

// Mostrar la cadena JSON
console.log(jsonStr); // Salida: {"nombre":"Juan","edad":30,"ciudad":"Madrid"}

// Convertir una cadena JSON a un objeto JavaScript
let personaDesdeJson = JSON.parse(jsonStr);

// Mostrar el objeto JavaScript resultante
console.log(personaDesdeJson); // Salida: { nombre: 'Juan', edad: 30, ciudad: 'Madrid' }
