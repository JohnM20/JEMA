Javascript


Javascript es un lenguaje de programación el cual le añade interactividad a las páginas WEB, a continuación se ejecuta un codigo básico el cual genera una alerta en el navegador


		
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Alerta "Hola Mundo"</title>
</head>
<body>
    <script>
            alert("¡Hola Mundo!");     
    </script>
</body>
</html>

Variables en Javascript


Las variables en javascript se guardan según su caracter de ser fijas durante todo el programa const o que varien su valor let, así como los datos que guarda, si son de tipo númerico o caracteres. A continuación se presentan diversos ejemplos de variables:


    

let entero = 10; // Variable entera
const pi = 3.14; // Variable decimal
const cadena = "Hola Mundo"; // Variable de cadena (string)
let booleano = true; // Variable booleana (true o false)

Operaciones con variables numericas


Las variables numericas pueden ser modificables y operables según los operadores matemáticos que se requieran, a continuación se muestra un ejemplo:


    

// Declaración de variables
let numero1 = 10;
let numero2 = 5;

// Operaciones matemáticas
let suma = numero1 + numero2;
let resta = numero1 - numero2;
let multiplicacion = numero1 * numero2;
let division = numero1 / numero2;
let modulo = numero1 % numero2; // Operación de módulo (resto de la división)

Visualización de variables en javascript


En javascript, se puede mostrar la información de las variables usando la consola con el comando console.log, modificando las etiquetas HTML o usando las alertas como se puede visualizar el siguiente codigo: 


    

// Declaración de variables
let numero1 = 10;
let numero2 = 5;

// Operación matemática
let resultado = numero1 * numero2; // Multiplicación de 10 por 5

// Mostrar el resultado usando un alert
alert("El resultado de la multiplicación es: " + resultado);

Obtener variables en Javascript


Para obtener variables en Javascript se puede realizar desde las etiquetas HTML, o con los comandos prompt o confirm(si son de caracter booleano). A continuación se muestra un ejemplo:


    

// Pedir al usuario que ingrese los valores mediante prompts
let numero1 = prompt("Ingrese el primer número:");
let numero2 = prompt("Ingrese el segundo número:");

// Convertir los valores ingresados a números
numero1 = parseFloat(numero1);
numero2 = parseFloat(numero2);

// Operación matemática
let resultado = numero1 * numero2; // Multiplicación de los números ingresados

// Mostrar el resultado usando un alert
alert("El resultado de la multiplicación es: " + resultado);
