Tablas


En ocasiones es necesario generar tablas, para esta finalidad se usan las etiquetas TR y TD.


Formularios


Las etiquetas input permiten ingresar los datos a la página WEB, los más empleados son:


text:Cuadro de texto

radio button:Seleccionar entre varias opciones

Checkbox:Seleccionar entre varias opciones

Select: Seleccionar entre varias opciones.


Árbol DOM



El árbol DOM es la estructura jerarquica de una página WEB, la mayoria de elementos WEB se encuentran dentro de etiquetas como se muestra en la imagen anterior, y según este conformado un elemento dentro de otro elemento se genera lo que visualmente denominamos página WEB, así también modificamos su contenido.
