Texto


Las etiquetas más fundamentales de HTML son y en las etiquetas head se escribe toda la información de la página respecto a su configuración y en la etiqueta body todo el contenido. Para generar un texto son las siguientes etiquetas:


P:Parrafo

H1: Texto más grande

H2: Texto más pequeño

br: salto de linea


Tablas


En ocasiones es necesario generar tablas, para esta finalidad se usan las etiquetas TR y TD.


Formularios


Las etiquetas input permiten ingresar los datos a la página WEB, los más empleados son:


text:Cuadro de texto

radio button:Seleccionar entre varias opciones

Checkbox:Seleccionar entre varias opciones

Select: Seleccionar entre varias opciones.
