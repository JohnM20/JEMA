CSS Grid

Las cuadriculas nos permiten organizar elementos de forma eficiente, esta css grid y flex box que nos permite alinear elementos de la mejor manera. El termino maquetación hace referencia al diseño de la página, donde una forma de hacerlo es dividir por "compartimientos" donde se publica la información. Estos compartimientos se llaman divisiones y con css grid podemos hacer realidad el diseño.

Es importante entender que toda la programación que realicemos con la tecnología frontend se facilmente portable porque se ejecutan desde cualquier computador, sin embargo las tecnologías backend unicamente se programan en servidores así que es necesario volver un equipo servidor para poder programar unicamente en el o contratar un servidor para poder programar la gestión y recepción de datos. Actualmente desde las tecnologías frontend se realizan multiples procesos para no cargar los servidores gracias a las altas prestaciones computacionales de los equipos empleando tecnologías como node.
