GIFS con GIMP


Usando Gimp apartir de varias imagenes, es posible definir cuanto tiempo se muestra cada uno creando así GIFS de forma rápida. Es muy util cuando se emplea la herramienta clonaciónporque permite duplicar la imagen generando pequeños cambios para que se vea la animación.



Como podemos ver en la imagen anterior en la parte derecha esta el panel de imagenes con el cual podemos realizar los diferentes cambios.
