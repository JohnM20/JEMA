Programación orientada a objetos


En el ámbito de la programación, existen varios paradigmas que definen cómo se escribe y organiza el código. Uno de los enfoques tradicionales es la programación estructurada, que se basa en seguir un flujo secuencial de instrucciones. En el desarrollo del curso de Python, este paradigma es ampliamente utilizado. Otro enfoque es la programación funcional, que organiza el código en bloques funcionales independientes que realizan tareas específicas. Finalmente, la programación orientada a objetos es un paradigma que organiza el código en torno a objetos y clases, como se ilustra en la siguiente imagen muestra cómo la programación orientada a objetos utiliza conceptos como clases y objetos usando como ejemplos los perros, en donde si bien todos son diferentes tienen unas caracteristicas unicas.


Sistema de Gestión de Biblioteca

El siguiente código Python implementa un sencillo sistema de gestión de biblioteca utilizando la programación orientada a objetos (POO). La POO es un paradigma que organiza el código en torno a "objetos" y "clases", facilitando la modularidad y reutilización del código.

Código Python


class Libro:
    def __init__(self, titulo, autor):
        self.titulo = titulo
        self.autor = autor
        self.disponible = True
    
    def cambiar_estado(self):
        self.disponible = not self.disponible
    
    def __str__(self):
        estado = "Disponible" if self.disponible else "Prestado"
        return f"{self.titulo} por {self.autor} - {estado}"

class Biblioteca:
    def __init__(self):
        self.libros = []
    
    def agregar_libro(self, libro):
        self.libros.append(libro)
    
    def prestar_libro(self, titulo):
        for libro in self.libros:
            if libro.titulo == titulo:
                if libro.disponible:
                    libro.cambiar_estado()
                    print(f"El libro '{titulo}' ha sido prestado.")
                else:
                    print(f"El libro '{titulo}' ya está prestado.")
                return
        print(f"El libro '{titulo}' no está disponible en la biblioteca.")
    
    def consultar_inventario(self):
        for libro in self.libros:
            print(libro)

# Uso del sistema
biblioteca = Biblioteca()
libro1 = Libro("1984", "George Orwell")
libro2 = Libro("To Kill a Mockingbird", "Harper Lee")

biblioteca.agregar_libro(libro1)
biblioteca.agregar_libro(libro2)

print("Inventario de la biblioteca:")
biblioteca.consultar_inventario()

print("\nPrestando '1984':")
biblioteca.prestar_libro("1984")

print("\nInventario después del préstamo:")
biblioteca.consultar_inventario()

print("\nIntentando prestar '1984' nuevamente:")
biblioteca.prestar_libro("1984")

Explicación de la Programación Orientada a Objetos

En este código:

  • Clase Libro: Representa un libro con atributos como título, autor y estado. Incluye métodos para cambiar el estado del libro y para representar el libro en una cadena.
  • Clase Biblioteca: Administra una colección de libros. Incluye métodos para agregar libros, prestar libros y consultar el inventario.

La programación orientada a objetos permite organizar el código en clases y objetos que interactúan entre sí. Esto facilita la creación de sistemas modulares y reutilizables, como el sistema de gestión de biblioteca mostrado aquí.
