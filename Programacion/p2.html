Python


Python es uno de los lenguajes de programación más populares y versátiles en el mundo de la tecnología. Su sintaxis sencilla y clara lo convierte en una excelente opción para principiantes, mientras que su robustez lo hace valioso para desarrolladores experimentados. Aprender Python no solo te permite escribir código de manera más rápida y eficiente, sino que también abre la puerta a una amplia gama de aplicaciones, desde desarrollo web y análisis de datos hasta inteligencia artificial y automatización.

Para entender por qué Python es tan especial, imagina que estás armando un rompecabezas. Python es como un rompecabezas con piezas grandes y coloridas que encajan fácilmente, lo que facilita que los principiantes comiencen a armarlo. La sintaxis clara y la estructura intuitiva de Python permiten que te concentres en resolver el problema en lugar de luchar con la complejidad del lenguaje.

Además, Python es como una caja de herramientas multifunción. En lugar de tener que buscar diferentes herramientas para cada tarea, Python te ofrece una sola herramienta que puede hacer muchas cosas, desde manejar datos y crear aplicaciones web hasta desarrollar inteligencia artificial. Esta versatilidad es una de las razones por las que es tan popular entre los desarrolladores.

Por último, aprender Python te prepara para enfrentar problemas complejos con soluciones elegantes y efectivas. Es como tener una brújula que te guía en un viaje, ayudándote a encontrar el camino más claro y directo hacia tus objetivos, ya sea en el mercado laboral o en proyectos personales.

Una variable es un concepto abstracto de algo que tiene ciertas características, por ejemplo, si pensamos en una silla ideamos un objeto quizás con 4 patas un espaldar donde su principal función es permitir el descanso de las personas. print("ingrese la primera nota");
n1=float(input());

nf=(n1+n2)/2;
print("su definitiva es");
print(nf);

Se puede concluir que es necesario trabajar con variables porque los valores siempre van a ser diferentes. Sin embargo, debes tener en cuenta lo siguiente:

Print

Cuando se va a mostrar un valor de texto se debe poner "" en el programa con en este caso "ingrese la primera nota", cuando sea un valor de una variable no es necesario como en el ejemplo es n2.

input()

Antes del input existe una palabra denominada float en ocasiones también int, esto quiere decir que se espera una variable numérica lo que implica que solamente se van a almacenar números. Cuando se quiere almacenar un texto no es necesario poner ninguna palabra previa. Se puede seguir el ejemplo su uso: 


			
print("ingrese su primer nombre");
n1=input();
print("ingrese su primer apellido");
n2=input();
print("Bienvenido");
print(n1);
print(n2);
