Condicionales en Python


Las condicionales en Python manejan unos espacios y puntuaciones que se deben seguir para que funcione el programa, así también tenemos que saber que si queremos comparar con un valor puntual debemos emplear ==(doble igual) como se muestra en el siguiente programa que es el programa anterior escrito en Python: 

    
print("ingrese el clima ¿sol o lluvia?");
clima=input();
if clima=="lluvia":
    print("lleva sombrilla");
else:
    print("lleva gafas de sol");

Operadores de relación


Dependiendo la condición que necesitemos comparar se emplean después del if, distintos tipos de operadores de relación por ejemplo si necesitamos comparar una condición donde un número X sea mayor a 5, se escribiría de la siguiente manera X>5, si es un intervalo donde sean números mayores a 5 y menores a 10, se escribiría de la siguiente manera X>5 AND X<10 .


Para finalizar si se requiere comparar con un valor puntual, como es por ejemplo si el número X debe ser igual a 8 para que se cumpla la condición se escribiria de la siguiente manera X==8
