Listas


Las listas en Python son una forma de almacenar diferentes datos en una misma variable. A continuación, se muestran varios ejemplos:


Numeros_pares = [0, 2, 4, 6, 8, 10]
Numeros_primos = [1, 2, 3, 5, 7, 11]
Colores = ["rojo", "azul", "gris", "blanco", "negro"]
Ingredientes_ensalada = ["lechuga", "tomate", "pepino", "aceite", "zanahoria"]

Para ilustrar cómo trabajar con listas, guardaremos los nombres de 6 estudiantes de un curso. Esto se haría de la siguiente manera:


Ejemplo 1: Acceder a Elementos de la Lista


# Lista de números pares
Numeros_pares = [0, 2, 4, 6, 8, 10]

# Acceder al primer elemento
print(Numeros_pares[0])  # Imprime: 0

# Acceder al cuarto elemento
print(Numeros_pares[3])  # Imprime: 6

Ejemplo 2: Modificar un Elemento de la Lista


# Lista de colores
Colores = ["rojo", "azul", "gris", "blanco", "negro"]

# Modificar el tercer elemento
Colores[2] = "verde"
print(Colores)  # Imprime: ['rojo', 'azul', 'verde', 'blanco', 'negro']

Ejemplo 3: Añadir Elementos a la Lista


# Lista de ingredientes para ensalada
Ingredientes_ensalada = ["lechuga", "tomate", "pepino", "aceite", "zanahoria"]

# Añadir un nuevo ingrediente
Ingredientes_ensalada.append("aceitunas")
print(Ingredientes_ensalada)  # Imprime: ['lechuga', 'tomate', 'pepino', 'aceite', 'zanahoria', 'aceitunas']

Ejemplo 4: Insertar Elementos en una Posición Específica


# Lista de números primos
Numeros_primos = [1, 2, 3, 5, 7, 11]

# Insertar un nuevo número en la segunda posición
Numeros_primos.insert(1, 4)
print(Numeros_primos)  # Imprime: [1, 4, 2, 3, 5, 7, 11]

Ejemplo 5: Eliminar Elementos de la Lista


# Lista de colores
Colores = ["rojo", "azul", "verde", "blanco", "negro"]

# Eliminar un color específico
Colores.remove("blanco")
print(Colores)  # Imprime: ['rojo', 'azul', 'verde', 'negro']

# Eliminar el último color
ultimo_color = Colores.pop()
print(ultimo_color)  # Imprime: negro
print(Colores)  # Imprime: ['rojo', 'azul', 'verde']

Ejemplo 6: Ordenar una Lista


# Lista de números pares
Numeros_pares = [0, 2, 4, 6, 8, 10]

# Ordenar la lista en orden ascendente
Numeros_pares.sort()
print(Numeros_pares)  # Imprime: [0, 2, 4, 6, 8, 10]

# Ordenar en orden descendente
Numeros_pares.sort(reverse=True)
print(Numeros_pares)  # Imprime: [10, 8, 6, 4, 2, 0]

Ejemplo 7: Contar Elementos en una Lista


# Lista de ingredientes para ensalada con repetidos
Ingredientes_ensalada = ["lechuga", "tomate", "pepino", "aceite", "zanahoria", "tomate"]

# Contar cuántas veces aparece 'tomate'
cantidad_tomates = Ingredientes_ensalada.count("tomate")
print(cantidad_tomates)  # Imprime: 2

Ejemplo 8: Obtener la Longitud de la Lista


# Lista de nombres de estudiantes
Nombres_estudiantes = ["Ana", "Luis", "Pedro", "Sofia", "María", "Juan"]

# Obtener la longitud de la lista
numero_estudiantes = len(Nombres_estudiantes)
print(numero_estudiantes)  # Imprime: 6

Listas con variables

+

En Python, puedes crear listas usando variables, lo cual es muy útil cuando los datos de la lista provienen de cálculos, entradas del usuario o cualquier otra fuente dinámica.

# Variables
			elemento1 = "manzana"
			elemento2 = "banana"
			elemento3 = "cereza"
			
			# Crear lista con variables
			frutas = [elemento1, elemento2, elemento3]
			
			print(frutas)  # Imprime: ['manzana', 'banana', 'cereza']

Libreria matplotlib


La libreria matplotlib permite apartir de listas generar diversos gráficos, a continuación se presenta un ejemplo con su respectivo gráfico:



import matplotlib.pyplot as plt

# Ejemplo de gráfico de líneas
x = [1, 2, 3, 4]
y = [10, 20, 15, 30]

plt.plot(x, y)
plt.title("Ejemplo de Gráfico de Líneas")
plt.xlabel("Tiempo")
plt.ylabel("Nivel de Energía")
plt.show()

Gráficas de barras y circulares

+

Para generar una gráfica de barras se utiliza el siguiente código:



import matplotlib.pyplot as plt

# Datos del estado de ánimo durante la semana
dias = ["Lunes", "Martes", "Miércoles", "Jueves", "Viernes", "Sábado", "Domingo"]
estado_animo = [7, 5, 6, 8, 4, 7, 6]

plt.bar(dias, estado_animo, color='skyblue')
plt.title("Estado de Ánimo Semanal")
plt.xlabel("Días de la Semana")
plt.ylabel("Nivel de Ánimo (1-10)")
plt.show()

Para generar un diagrama circular se utiliza el siguiente código:



import matplotlib.pyplot as plt

# Actividades del día y el tiempo dedicado
actividades = ["Estudio", "Descanso", "Ejercicio", "Tiempo Libre"]
tiempo_dedicado = [5, 3, 1, 2]

plt.pie(tiempo_dedicado, labels=actividades, autopct="%1.1f%%", colors=['lightcoral', 'lightblue', 'yellowgreen', 'violet'])
plt.title("Distribución del Tiempo en Actividades Diarias")
plt.show()
