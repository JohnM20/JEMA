Flask

Flask es un microframework para Python que permite crear aplicaciones web de manera sencilla y rápida. Es ideal para proyectos pequeños y prototipos.

Ejemplo de Flask

from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "¡Bienvenido a mi aplicación Flask!"

if __name__ == '__main__':
    app.run(debug=True)

Django

Django es un framework web de alto nivel que fomenta el desarrollo rápido de aplicaciones web. Proporciona muchas funcionalidades integradas.

Ejemplo de Django

# En la terminal
django-admin startproject mi_proyecto
cd mi_proyecto
python manage.py startapp mi_aplicacion

# views.py en mi_aplicacion
from django.http import HttpResponse

def home(request):
    return HttpResponse("¡Bienvenido a mi aplicación Django!")

Pygame

Pygame es una biblioteca diseñada para crear videojuegos en Python. Proporciona herramientas para gestionar gráficos y sonido.

Ejemplo de Pygame

import pygame
import sys

pygame.init()

# Configurar la ventana
screen = pygame.display.set_mode((400, 300))
pygame.display.set_caption('Hola Pygame')

# Bucle principal
while True:
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            pygame.quit()
            sys.exit()
    
    screen.fill((255, 255, 255))  # Fondo blanco
    pygame.display.flip()
