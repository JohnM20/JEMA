Operaciones Booleanas en 3D en OpenSCAD


Las operaciones booleanas permiten combinar formas 3D mediante unión, diferencia e intersección.


1. Unión (union())

Permite combinar dos o más objetos en uno solo.

union() {
    cube([20, 20, 20]);
    translate([10, 10, 0]) sphere(15);
}

2. Diferencia (difference())

Resta un objeto de otro, creando cortes y formas huecas.

difference() {
    cube([30, 30, 30]);
    translate([5, 5, -5]) sphere(20);
}

3. Intersección (intersection())

Solo mantiene la parte en la que los objetos se superponen.

intersection() {
    cube([30, 30, 30]);
    translate([10, 10, 0]) sphere(20);
}

4. Uso Combinado

Podemos combinar múltiples operaciones booleanas para crear diseños más complejos.

difference() {
    union() {
        cube([30, 30, 30]);
        translate([15, 15, 0]) cylinder(30, 10, 10);
    }
    translate([5, 5, 5]) sphere(15);
}
