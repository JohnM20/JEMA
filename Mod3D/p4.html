Operaciones Booleanas en 2D


Las operaciones booleanas en OpenSCAD permiten combinar o modificar formas geométricas utilizando unión, diferencia e intersección.


1. Unión (union())

Combina dos o más objetos en uno solo.

union() {
    square([30, 30]);
    circle(20);
}

2. Diferencia (difference())

Resta un objeto de otro.

difference() {
    square([40, 40]);
    circle(20);
}

3. Intersección (intersection())

Deja solo la parte común entre dos o más objetos.

intersection() {
    square([40, 40]);
    circle(30);
}

4. Uso de Diferencia para Crear una Figura Hueca

difference() {
    circle(30);
    circle(20);
}

5. Creación de un Semicírculo

difference() {
    circle(30);
    square([60, 30]);
}

Estas operaciones permiten generar formas complejas combinando figuras básicas.
