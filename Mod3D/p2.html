Coordenadas 2D y Figuras Básicas en OpenSCAD


En OpenSCAD, el sistema de coordenadas 2D se basa en ejes (x, y). Se pueden crear figuras básicas como cuadrados, rectángulos y círculos mediante funciones predefinidas.


Figuras Básicas en OpenSCAD


Cuadrado y Rectángulo:

square([width, height]); // Crea un rectángulo o cuadrado
square(50); // Crea un cuadrado de 50x50 unidades
square([80, 40]); // Crea un rectángulo de 80 de ancho y 40 de alto

El parámetro center=true centra la figura en el origen.


Círculo:

circle(r=30); // Crea un círculo con radio 30 unidades

El parámetro $fn permite ajustar la calidad del círculo.


Ejemplo de Uso de Polígonos


Podemos definir un polígono especificando una lista de puntos:

polygon(points=[[0,0], [100,0], [50,50], [0,100]]);

Esto generará un cuadrilátero con vértices en las coordenadas indicadas.


Generación de Polígonos Regulares a partir de Cilindros


Podemos generar polígonos regulares usando cilindros con pocos segmentos:

cylinder(h=10, r=50, $fn=6); // Genera un hexágono

El parámetro $fn determina el número de lados del polígono resultante.
