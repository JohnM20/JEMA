Módulos en OpenSCAD


Los módulos en OpenSCAD permiten reutilizar código y organizar proyectos de manera eficiente. Se usan para definir formas y estructuras que pueden ser invocadas varias veces con diferentes parámetros.


1. Creación de un módulo básico

Definimos un módulo con la palabra clave module seguido del nombre y sus parámetros.

module cilindro_base(altura, radio) {
    cylinder(h=altura, r=radio);
}

cilindro_base(50, 10); // Llamada al módulo

2. Uso de módulos dentro de otros módulos

Podemos anidar módulos para construir estructuras más complejas.

module base() {
    cube([30, 30, 5]);
}

module soporte() {
    translate([15, 15, 5]) cylinder(h=40, r=5);
}

module estructura_completa() {
    base();
    soporte();
}

estructura_completa(); // Genera la estructura

3. Parámetros para personalizar módulos

Los módulos pueden recibir parámetros que permiten modificar sus características.

module caja(ancho, largo, alto) {
    cube([ancho, largo, alto]);
}

caja(40, 20, 10);
caja(20, 20, 20);

4. Uso en la creación de ensamblajes

Los módulos son útiles para crear partes reutilizables en diseños más complejos.

module pata() {
    cylinder(h=50, r=5);
}

module mesa() {
    cube([60, 40, 5]);
    translate([5, 5, -50]) pata();
    translate([55, 5, -50]) pata();
    translate([5, 35, -50]) pata();
    translate([55, 35, -50]) pata();
}

mesa();

Los módulos permiten estructurar y reutilizar código para crear modelos más eficientes y organizados.
