Operaciones con Geometría 3D en OpenSCAD


En OpenSCAD, se pueden realizar operaciones booleanas con objetos 3D para crear formas complejas a partir de primitivas.


1. Unión (union())

Combina dos o más objetos en una sola forma.

union() {
    cube([30, 30, 30]);
    translate([15, 15, 0]) sphere(20);
}

2. Diferencia (difference())

Resta un objeto de otro, dejando la parte restante.

difference() {
    cube([40, 40, 40]);
    translate([10, 10, 10]) sphere(25);
}

3. Intersección (intersection())

Genera la parte común entre dos objetos.

intersection() {
    cube([40, 40, 40]);
    translate([10, 10, 10]) sphere(25);
}

Estas operaciones permiten construir formas más complejas combinando elementos básicos.
