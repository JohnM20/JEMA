Loops en OpenSCAD


Los bucles (for) en OpenSCAD permiten repetir instrucciones para generar patrones o estructuras complejas de manera eficiente.


1. Uso de un loop básico

Ejemplo de un bucle que genera una serie de cilindros en una línea.

for (i = [0:5]) {
    translate([i * 10, 0, 0])
    cylinder(h=10, r=3);
}

2. Bucles anidados

Podemos usar loops dentro de otros para generar estructuras en cuadrícula.

for (x = [0:3]) {
    for (y = [0:3]) {
        translate([x * 10, y * 10, 0])
        cube([5, 5, 5]);
    }
}

3. Uso de loops con transformaciones

Ejemplo de un bucle que genera un patrón circular con cilindros.

for (i = [0:5]) {
    rotate([0, 0, i * 60])
    translate([20, 0, 0])
    cylinder(h=10, r=3);
}

4. Uso de loops para generar estructuras complejas

Ejemplo de un bucle que crea una escalera.

for (i = [0:5]) {
    translate([i * 10, 0, i * 5])
    cube([10, 20, 5]);
}

Los loops permiten automatizar la creación de modelos y optimizar el código en OpenSCAD.
