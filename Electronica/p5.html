Fuentes de energía electríca


Una vez es claro que existe la energía electríca alterna y que tiene un proceso para volverse en continua, la aplicación que vamos a ver a continuación es para poder adaptar el voltaje o la corriente continua según se requiera. El voltaje continuo podemos obtenerlo de un proceso de rectificación y regulación desde la toma como un adaptador de celular o desde las baterias donde especifica cuanto voltaje genera así como la corriente.



En el caso de las baterias la cantidad de electrones se pierden cada vez que se emplea y cada vez se desgastan más a excepción que sean recargables las cuales tienen más tiempo de utilidad, por este motivo se especifica el voltaje inicial en la mayoria de las baterias pero es una cantidad que varia y la corriente se mide por Amperios-Hora.


Suma de fuentes para obtener voltaje


En ocasiones el voltaje a emplear no es suficiente y necesitamos añadir más fuentes de voltaje, una forma de solucionarlo es sumar las fuentes siempre y cuando se comparta la misma corriente en las fuentes de voltaje. Este sistema lo podemos observar en las pilas de los controles remotos que empleamos diariamente donde las conectamos en serie para obtener más voltaje.


Divisor de voltaje

Este circuito que se presenta funciona para dividir el voltaje según las necesidades y la corriente es la misma en todo el circuito. Así por ejemplo si tenemos un sensor que funciona unicamente con 5V, y tenemos un cargador de celular de 12V podemos ajustar las resistencias para obtener el voltaje necesario.

Suma de fuentes para obtener corriente

Para sumar fuentes de corriente se deben configurar en paralelo siempre y cuando se compartan los mismos voltajes en las diversas fuentes.


Divisor de corriente

Los divisores de corriente permiten dividir la corriente según las necesidades que tengamos en el circuito, así por ejemplo si tenemos una fuente de 5v pero con más corriente de la necesaria con esta configuración podemos dividir la corriente por las resistencias para tener la corriente y el voltaje adecuados.
