Filtros


Según la acumulación de bits de una imagen se genera frecuencias si se repiten la información. Si se deja pasar muchas frecuencias se denomina filtros pasaaltas, asi también si solo son frecuencias medias se denomina filtros pasabandasy si solamente se emplean las frecuencias bajas se denomina frecuencias pasabajas A continuación se presenta una imagén del resultado de aplicar filtros:
