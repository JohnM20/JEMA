Manipulación de imagenes


La finalidad de un filtro digital es generar en la imagen correcciones que no se realizaron al momento de elaborarse de una tomafotográfica o escaner. Así también permite generar nuevas caracteristicas una caracteristica primordial es eliminar el fondo de una imagen para resaltar o reemplazar una imagen (unsharp mask=mascara de desenfoque) en otra como se muestra a continuación:



Desenfoque de una imagen


El desenfoque permite entre muchas opciones, hacer un contraste para resaltar visualmente una imagen como se muestra a continuación:



Filtros


Según la acumulación de bits de una imagen se genera frecuencias si se repiten la información. Si se deja pasar muchas frecuencias se denomina filtros pasaaltas, asi también si solo son frecuencias medias se denomina filtros pasabandasy si solamente se emplean las frecuencias bajas se denomina frecuencias pasabajas A continuación se presenta una imagén del resultado de aplicar filtros:
